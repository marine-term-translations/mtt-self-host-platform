## Interaction States

### Hover States
- Language cards: subtle border color change
- Add button: border changes to marine blue, background tint
- Move buttons: light gray background
- Remove button: light red background
- Modal language items: marine blue border and background tint

### Disabled States
- Move up on first item: opacity 30%, no hover effect
- Move down on last item: opacity 30%, no hover effect
- Save button when no changes: gray background, cursor not-allowed

### Loading States
- Initial load: spinning loader with "Loading languages..."
- Saving: button shows spinner and "Saving..."
